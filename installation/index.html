Noxic's Discord bot template

Developer Setup Guide

1

Clone Repository

Clone the bot's source code from GitHub:

git clone https://github.com/noxic-dev/empty-discord-bot.git
cd empty-discord-bot
Prerequisites: BunJS V1.2.1+ and Git installed
2

Configure config.json

Create and modify the configuration file:

        
{
    "botStatus": "Bot made with template by Noxics"
}
vim config.json
3

Set Up .env File

Create environment variables for sensitive data:

TOKEN=YOUR_BOT_TOKEN_HERE
