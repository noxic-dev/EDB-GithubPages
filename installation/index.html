Installation Guide

1

Invite the Bot

Click the invite button below to add NeonBot to your Discord server:

Note: Requires "Manage Server" permissions
2

Set Up Permissions

After inviting, configure the bot's permissions:

/setup-permissions administrator

Or specify custom permissions:

/setup-permissions manage_roles send_messages
3

Configuration

Set your server's default configuration:

/config set prefix=!
/config set language=en
Pro Tip: Use /config help for all configuration options
4

Verification

Confirm the bot is working properly:

/ping
